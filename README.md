# Macys-Backbone-Demo-App



## Installation & Usage

```
npm install

npm run build

Browse build/index.html to run the application.

Concepts are implemented (based on Marionette documenatation)

> backbone.radio
> marionette.routing
> marionette.routing >> sub router link
> rivet.databinding / marionette.behaviors
> region/child-view render
> routing.context

Things can be enhanced:
> code-splitting, tree shaking, lazing loading, router based loading via webpack
> Performance improvements
> Componentize the view, modularize the code in more granular. 
> Node server implementation
> SASSS implementation.

```
